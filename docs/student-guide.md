# JavaEdu Student Guide

Welcome to JavaEdu! This guide will help you use the platform to learn Java programming.

## Getting Started

### Installing the Eclipse Plugin

1. Download the JavaEdu plugin file (`.jar`)
2. Place it in your Eclipse `dropins` folder:
   - Windows: `C:\Users\<username>\eclipse\dropins`
   - Mac: `~/eclipse/dropins`
   - Linux: `~/eclipse/dropins`
3. Restart Eclipse
4. Look for the "JavaEdu" menu in Eclipse

### Logging In

1. In Eclipse, go to **JavaEdu > Login...**
2. Enter your email and password
3. Click **Login**

If you don't have an account:
1. Click "New user? Register here"
2. Enter your name, email, and password
3. Click **Register**

## Using the Plugin

### Viewing Exercises

1. Open **Window > Show View > Other**
2. Select **JavaEdu > Exercises**
3. The Exercise List view shows all available exercises

Exercises are organized by course. Click a course to expand it and see its exercises.

### Understanding Exercise Details

When you select an exercise, you'll see:
- **Title**: Name of the exercise
- **Difficulty**: How challenging it is (Beginner to Expert)
- **Points**: Maximum points you can earn
- **Due Date**: When the exercise is due (if set)
- **Description**: What you need to do
- **Starter Code**: Code to begin with

### Working on an Exercise

1. **Read the description carefully**
   - Understand what the exercise asks
   - Note any constraints or requirements

2. **Copy the starter code**
   - Click "Copy to Clipboard" in the Exercise Details view
   - Create a new Java file in your workspace
   - Paste the starter code

3. **Write your solution**
   - Use Eclipse's features (autocomplete, debugging)
   - Test your code locally first

4. **Submit when ready**
   - Go to **JavaEdu > Submit Solution**
   - Or use the toolbar button

## Getting Help

### Using Hints

If you're stuck:
1. Open the **Exercise Details** view
2. Click the **Hints** tab
3. Click "Reveal Next Hint"

Hints are designed to guide you without giving away the answer. Use them wisely!

### AI Helper

The AI Helper can answer questions about your code:

1. Open **Window > Show View > Other > JavaEdu > AI Helper**
2. Type your question
3. Check "Include my current code" to share your code for context
4. Click **Ask**

**Important**: The AI tutor will:
- Help you understand concepts
- Point you in the right direction
- Suggest debugging approaches

The AI tutor will NOT:
- Give you the answer directly
- Write your code for you

**Rate limits**: You have a limited number of AI questions per hour. Use them thoughtfully.

### Common Questions for the AI

Good questions:
- "Why is my loop running forever?"
- "What does NullPointerException mean?"
- "How do I convert a String to an int?"
- "Can you explain how arrays work?"

Less helpful questions:
- "What's the answer?"
- "Write the code for me"
- "Just fix it"

## Submitting Solutions

### Before Submitting

1. **Test locally**: Run your code to check for obvious errors
2. **Check requirements**: Re-read the exercise description
3. **Review your code**: Look for typos and logic errors

### Submitting

1. Make sure your solution file is open in the editor
2. Select the exercise in the Exercise List view
3. Click **Submit Solution** in the toolbar or menu
4. Wait for the results

### Understanding Results

After submission, you'll see:

- **Score**: Points earned out of maximum
- **Status**: COMPLETED, COMPILATION_ERROR, RUNTIME_ERROR, or TIMEOUT
- **Test Results**: Which tests passed or failed

#### Status Meanings

- **COMPLETED**: Your code compiled and ran. Check the score.
- **COMPILATION_ERROR**: Your code has syntax errors. Check the error message.
- **RUNTIME_ERROR**: Your code crashed during execution (e.g., NullPointerException)
- **TIMEOUT**: Your code took too long. You might have an infinite loop.

### Improving Your Score

You can submit multiple times. Tips for improvement:
- Read the error messages carefully
- Use hints if available
- Test with different inputs locally
- Check edge cases

## Viewing Grades

To see all your grades:
1. Open **Window > Show View > Other > JavaEdu > My Grades**
2. See all your submissions and scores
3. Click a submission to view feedback

## Tips for Success

### Time Management

- Start exercises early
- Don't wait until the due date
- Allow time for debugging

### Learning Effectively

- Read the description completely before coding
- Try to solve it yourself before using hints
- Learn from mistakes in failed tests

### Good Coding Habits

- Use meaningful variable names
- Add comments to explain complex logic
- Test your code with different inputs
- Handle edge cases

### Getting Unstuck

1. Read the error message carefully
2. Check if you followed the requirements
3. Use debugging (print statements, breakpoints)
4. Reveal a hint
5. Ask the AI Helper
6. Ask your teacher

## Troubleshooting

### Can't log in

- Check your email and password
- Make sure you registered first
- Contact your teacher if issues persist

### Exercises not showing

- Make sure you're logged in
- Check that you're enrolled in the course
- Try refreshing (JavaEdu > Refresh Exercises)

### Submission stuck on "Pending"

- Wait a few seconds; tests take time
- If it stays stuck, try submitting again
- Check your internet connection

### "Compilation Error" but code works locally

- Make sure your class name matches the file name
- Check that you're using Java 8 compatible code
- Look for any IDE-specific features that might not work

### Tests failing unexpectedly

- Check the exact expected output format
- Watch for trailing spaces or newlines
- Verify your method signatures match exactly
